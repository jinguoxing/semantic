# 语义协同中心 - 控制台维度方案与示例数据

## 1. 控制台命名
- 新名称: 语义协同中心

## 2. 维度方案（双视角协同）

### 2.1 业务视角
- 业务对象规模: 对象总数、已发布/草稿/归档
- 业务目标覆盖: 目标数、已绑定对象数、覆盖率
- KPI 与指标: KPI 总数、已定义口径数、口径一致率
- 业务场景落地: 场景总数、已编排/运行/草稿
- 业务价值进展: 高优先级目标完成率、关键对象热度

### 2.2 技术治理视角
- 数据资产接入: 数据源数、扫描表数、可用资产数
- 语义映射进度: 映射完成率、未映射对象数
- 质量与规则: 命名达标率、规则覆盖率、冲突数
- 变更与发布: 变更次数、发布成功率、回滚数
- 候选与审核: 候选生成数、确认通过率、平均确认周期

### 2.3 协同联动呈现
- 协同指标:
  - 目标覆盖率 = 目标绑定对象 / 已建对象
  - 口径一致率 = 已发布KPI口径 / KPI总数
  - 映射达标率 = 已映射对象 / 已发布对象
- 协同流程链:
  - 业务目标 -> KPI口径 -> 业务对象 -> 映射 -> 资产/服务
- 双视角同步卡片:
  - 左侧业务达成, 右侧治理保障, 中间协同指标

## 4. 组件级排版清单

### A. 顶部总览条
1) 对象规模
- 主值: 对象总数
- 辅值: 已发布 / 草稿 / 归档
- 图表: 迷你堆叠条

2) 候选与审核
- 主值: 候选总数
- 辅值: 待确认 / 已确认 / 忽略
- 图表: 环形占比

3) 冲突与风险
- 主值: 冲突数
- 辅值: 已解决 / 待处理
- 图表: 小型趋势折线

4) 协同成熟度指数
- 主值: 协同成熟度分（0-100）
- 构成: 目标覆盖率 x 口径一致率 x 映射达标率
- 图表: 仪表盘或进度环

### B. 业务视角
1) 业务目标覆盖
- 主值: 覆盖率
- 辅值: 已绑定目标 / 目标总数
- 图表: 进度条 + Top3 目标列表

2) KPI 与口径
- 主值: 口径一致率
- 辅值: 已定义口径 / KPI 总数
- 图表: 堆叠条 + 待定义列表

3) 场景落地
- 主值: 已编排场景数
- 辅值: 运行中 / 草稿
- 图表: 柱状（场景状态分布）

4) 关键对象热度
- 主值: TOP5 对象
- 辅值: 最近访问 / 变更次数
- 图表: 热度条形列表

### C. 技术治理视角
1) 资产接入
- 主值: 数据源数
- 辅值: 扫描表数 / 可用资产数
- 图表: 折线（接入增长）

2) 映射进度
- 主值: 映射完成率
- 辅值: 已映射对象 / 未映射对象
- 图表: 进度环 + 未映射Top5

3) 质量与规则
- 主值: 命名达标率
- 辅值: 规则覆盖率 / 冲突数
- 图表: 雷达图或三指标条形

4) 发布与变更
- 主值: 发布成功率
- 辅值: 变更次数 / 回滚次数
- 图表: 小型趋势折线

### D. 协同联动区
- 协同指标卡（3 列并排）
  - 目标覆盖率
  - 口径一致率
  - 映射达标率
- 协同流程链卡片
  - 文案: 业务目标 -> KPI口径 -> 业务对象 -> 映射 -> 资产/服务

### E. 风险与待办区
- 业务阻点卡: 未绑定目标 / 未定义口径 / 场景缺口
- 治理阻点卡: 未映射对象 / 冲突未解决 / 低质量对象
- 快捷处理入口: 去绑定目标 / 去定义口径 / 去处理映射
