
# 语义建模概览页

## Figma Frame 结构说明（Final）

> 页面目标：
> **一屏判断状态，两屏完成决策跳转**

---

## 0️⃣ 页面基本信息（Frame Meta）

* **Frame 名称**：`SemanticModelingOverview`
* **所属菜单**：语义治理 → 语义建模 → 语义建模概览
* **页面类型**：Dashboard / Overview
* **默认时间范围**：近 30 天（仅影响趋势类指标）

---

## 1️⃣ 顶部区域（Header Zone）

### Frame：`Header / PageTitle`

**结构**

```
[语义建模概览]        [时间范围切换] [状态标签]
```

**组件**

* Page Title（H1）：语义建模概览
* Subtitle（Body）：从业务与技术视角协同推进语义建模
* Time Range Switch：近 7 天 / 近 30 天 / 近 90 天
* Status Badge：`建模进行中 / 可进入下一阶段`

> ⚠️ 状态 Badge 由「建模阶段进展」综合判断，不可手工设置

### Frame：`Card / NextAction`

```

```

**规则**

* 永远高亮 1 个 CTA
* 高亮规则 = 当前阶段进展 + 阻塞判断

---

## 5️⃣ 设计与研发约束（必须写进 PRD）

### 设计约束

* 概览页 **不出现编辑控件**
* 所有卡片 **只跳转，不内嵌编辑**
* 指标异常用颜色，不用弹窗

### 研发约束

* 所有指标 **来自语义建模域数据**
* 不读取质量执行结果
* 不依赖语义版本是否发布
