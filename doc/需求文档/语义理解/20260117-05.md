# 语义理解证据页与结论页优化（20260117-05）

## 目标与范围
- 证据页补齐“冲突解释/反证结构”与“证据到动作闭环”。
- 语义结论页收敛对象类型、补齐“确认生效”动作与责任信息。
- 安全合规卡片与字段级敏感识别口径一致。

## 一、证据页优化（AI 证据 + 规则证据 + 权重可视化）

### 1. 冲突解释 + 反证结构
触发条件：AI 建议与规则/统计冲突时。
- 支持建议的证据（AI/规则）
- 反对建议的证据（Gate/规则拦截项）
- 融合说明：Rule Score / AI Score / Final（权重可视化）

### 2. 证据到动作闭环
证据卡片内置快捷动作：
- 接受建议
- 改判
- 标记待定

动作效果：
- 自动跳转字段裁决面板并定位字段
- 写入审计日志（动作/来源/字段/时间/理由）

### UI 结构（证据页）
- 证据链标题区（展开/收起）
- 冲突解释卡（支持证据/反对证据 + 融合说明）
- AI 证据卡（权重条 + 快捷动作）
- 规则证据卡（分组 + 快捷动作）

## 二、语义结论页优化（业务身份/对象类型/粒度 + 生命周期/质量/安全）

### 1. 对象类型信息架构收敛
对象类型限定为业务可理解的 3~4 类：
- 主体 / 过程 / 状态 / 清单
“规则/属性”改为标签/特征显示。

### 2. 确认生效（P0）
新增主按钮：确认生效（写入语义注册表）。
展示信息：
- 确认人
- 确认时间
- 生效范围
- 回滚入口

“加入候选业务对象”保留为草稿态。

### 3. 安全合规口径统一
- 表级安全等级 = 字段最高敏感等级
- 安全合规卡片可展开：敏感字段清单与证据来源（AI/规则）

### UI 结构（结论页）
- 业务身份区（业务名称/业务域/数据分层）
- 对象类型区（主体/过程/状态/清单 + 标签区）
- 数据粒度
- 三卡：生命周期 / 质量画像 / 安全合规（含明细展开）
- 确认生效信息卡（确认人/时间/范围/回滚入口）
- 操作区：确认生效 / 修正结果 / 加入候选业务对象

## 数据与日志
- 语义画像新增：confirmedBy / confirmedAt / confirmScope
- 审计日志新增：证据裁决日志（action/source/field/reason/time）

## 交互规则
- 证据动作触发后自动切换到字段裁决视图并定位
- 改判动作默认进入编辑状态
- 确认生效后治理状态为 S3（已确认生效）
