业务治理平台是一个企业级语义数据治理平台，提供从业务对象建模到数据映射、智能数据发现与分析的完整解决方案。

## ✨ 核心功能

### 🛒 数据服务超市

| 模块 | 功能描述 |
|------|----------|
| **问数** | 自然语言查询与对话式分析 |
| **找数** | 统一资产搜索、筛选与详情查看 |

### 📊 语义建模

| 模块 | 功能描述 |
|------|----------|
| **控制台** | 指标概览与关键流程入口 |
| **业务梳理** | 梳理业务目标、KPI、痛点与需求 |
| **业务对象建模** | 定义和管理业务对象及其字段结构 |
| **逻辑视图** | 结合扫描结果进行语义理解与逻辑建模 |
| **候选业务对象** | 候选对象确认与审核流程 |
| **场景编排** | 编排业务场景和数据流程 |

### 🧭 语义资产管理

| 模块 | 功能描述 |
|------|----------|
| **术语管理** | 术语库维护与标准化管理 |
| **标签管理** | 标签体系定义与业务标注 |

### 🔍 数据发现

| 模块 | 功能描述 |
|------|----------|
| **数据源管理** | 管理 MySQL、Oracle、PostgreSQL 等多种数据源 |
| **资产扫描** | 自动扫描数据库表结构和元数据 |

### ⚡ 服务执行

| 模块 | 功能描述 |
|------|----------|
| **API 网关** | 统一 API 入口和流量管理 |
| **缓存策略** | 智能缓存配置和优化 |

## 🎯 特色亮点

- **🛒 数据服务超市** - 找数与问数联动，覆盖搜索与对话式分析
- **🧭 语义资产管理** - 术语/标签双维度沉淀与复用
- **🌳 数据源树导航** - 按数据库类型分组展示，快速定位物理表
- **🎨 现代化 UI** - 响应式设计，侧边栏可收缩，流畅动画效果
- **📊 丰富可视化** - 柱状图、饼图、统计卡片等多种图表类型

## 🚀 快速开始

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看应用。

### 构建生产版本

```bash
npm run build
```

## 🛠 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式方案**: Tailwind CSS
- **图标库**: Lucide React
- **状态管理**: React Hooks (useState)
